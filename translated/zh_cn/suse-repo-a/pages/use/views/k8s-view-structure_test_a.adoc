== 概述

SUSE Observability 中的视图允许您监控和检查 IT 环境的子集。视图的结构是为有效过滤和可视化该子集（视图）中的数据而量身定制的。

=== 过滤器

通过视图用户界面右上角的 "筛选*器*"菜单，可以筛选视图中显示的组件（拓扑）、事件和轨迹。一旦应用，筛选器将影响视图中所有视角的内容。

* xref:/use/views/k8s-filters.adoc[过滤器]- 筛选视图中的组件（拓扑结构）、事件和轨迹

=== 视角

视图的*视角*以选项卡的形式显示在视图用户界面的左上角，可让您通过不同的视角直观地查看视图中的所有数据：

* xref:/use/views/k8s-overview-perspective.adoc[概览视角]或xref:/use/views/k8s-highlights-perspective.adoc[亮点视角]\--取决于您所处的视角类型
* xref:/use/views/k8s-topology-perspective.adoc[拓扑视角]\--视图组件的依赖关系图
* xref:/use/views/k8s-events-perspective.adoc[事件视角]- 拓扑上发生的所有事件
* xref:/use/views/k8s-metrics-perspective.adoc[指标视角]\--最相关组件的关键指标
* xref:/use/views/k8s-traces-perspective.adoc[跟踪视角]- 拓扑上运行的跟踪信息

[NOTE]
====
*所有视角都*将根据xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc[时间线]配置更新内容。
====
