= CHANGED Unterstützte Einheiten für Diagramme
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observabilität

Diese Seite enthält alle von Diagrammen unterstützten Einheiten. Für jede Einheit wird zunächst die Beschreibung der Einheit angegeben, gefolgt von dem Namen der Einheit, die in den SUSE Observability-APIs verwendet werden soll. Um zum Beispiel einen Wert in "Wissenschaftlicher Notation" anzuzeigen, verwenden Sie `+sci+` als Wert für die Einheit in einer metrischen Bindung.

Die Darstellung der Einheit passt sich automatisch an die Größe des Wertes an. Wenn die Einheit für ein Diagramm zum Beispiel `+bytes+` ist, wird der Wert als `+bytes(IEC)+` interpretiert. Ein Wert von `+100+` wird als `+100B+` wiedergegeben, ein Wert von `+5242880+` (`+5*1024*1024+` wird als `+5MiB+` angezeigt.
