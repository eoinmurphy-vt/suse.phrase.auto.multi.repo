== Visão geral

Uma visualização no SUSE Observability permite que você monitore e inspecione um subconjunto do seu ambiente de TI. A estrutura de uma visualização é adaptada para filtrar e visualizar os dados nesse subconjunto (visualização) de forma eficiente.

=== Filtros

O menu *Filtros*, no canto superior direito da interface do usuário da visualização, permite filtrar os componentes (topologia), eventos e traços exibidos em uma visualização. Uma vez aplicados, os filtros afetarão o conteúdo de todas as perspectivas em uma visualização.

* xref:/use/views/k8s-filters.adoc[Filtros] - Filtre os componentes (topologia), eventos e traços em sua visualização

=== Perspectivas

As *Perspectivas* de uma visualização são exibidas como guias no canto superior esquerdo da interface do usuário da visualização e permitem visualizar todos os dados em uma visualização por meio de diferentes lentes:

* xref:/use/views/k8s-overview-perspective.adoc[Perspectiva geral] ou xref:/use/views/k8s-highlights-perspective.adoc[Perspectiva de destaques] - dependendo do tipo de visualização em que você está
* xref:/use/views/k8s-topology-perspective.adoc[Perspectiva de topologia] - o mapa de dependência dos componentes da visualização
* xref:/use/views/k8s-events-perspective.adoc[Perspectiva de eventos] - todos os eventos que ocorrem na topologia
* xref:/use/views/k8s-metrics-perspective.adoc[Perspectiva das métricas] - principais métricas para os componentes mais relevantes
* xref:/use/views/k8s-traces-perspective.adoc[Perspectiva de] rastreamento - as informações de rastreamento em execução na topologia

[NOTE]
====
*Todas as perspectivas* atualizarão seu conteúdo com base na configuração xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc[da linha do tempo].
====
