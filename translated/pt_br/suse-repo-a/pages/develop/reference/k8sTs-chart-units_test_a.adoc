= Unidades suportadas para gráficos
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: Observabilidade do SUSE

Essa página contém todas as unidades suportadas pelos gráficos. Para cada unidade, primeiro é mencionada a descrição da unidade, seguida pelo nome da unidade que deve ser usada nas APIs do SUSE Observability. Por exemplo, para obter um valor exibido em "Notação científica", use `+sci+` como o valor da unidade em uma ligação métrica.

A renderização da unidade se ajusta automaticamente ao tamanho do valor. Por exemplo, se a unidade de um gráfico for `+bytes+`, o valor será interpretado como `+bytes(IEC)+`. Um valor de `+100+` será renderizado como `+100B+`, mas um valor de `+5242880+` (`+5*1024*1024+`) será mostrado como `+5MiB+`.
