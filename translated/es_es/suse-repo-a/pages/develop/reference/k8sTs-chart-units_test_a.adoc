= CAMBIADO Unidades admitidas para las cartas
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: Observabilidad de SUSE

Esta página contiene todas las unidades que admiten las cartas. Para cada unidad se menciona primero la descripción de la unidad, seguida del nombre de la unidad que debe utilizarse en las APIs de SUSE Observability. Por ejemplo, para obtener un valor mostrado en "Notación científica", utilice `+sci+` como valor para la unidad en un enlace métrico.

La representación de la unidad se ajusta automáticamente al tamaño del valor. Por ejemplo, si la unidad de un gráfico es `+bytes+`, el valor se interpreta como `+bytes(IEC)+`. Un valor de `+100+` se mostrará como `+100B+`, pero un valor de `+5242880+` (`+5*1024*1024+`) se mostrará como `+5MiB+`.
