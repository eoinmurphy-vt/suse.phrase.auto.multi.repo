= 关键角色及其作用
:revdate: 2025-05-30
:page-revdate: {revdate}

[cols="1,2,2", options="header"]
|===
|角色 |职责 |主要优势

|平台工程师
a|* 跨集群设置和管理{product_name} 安装。
* 定义并执行多集群应用和策略部署。
* 管理 GitRepo 和`fleet.yaml` 配置。
* 执行整个集群的更新和回滚。
a|* 跨集群集中控制
* 提高标准化和合规性。

|应用程序开发人员
a|* 在 Git 中构建应用程序 Helm 图表或清单。
* 在`fleet.yaml` 中定义部署目标和环境。
* 通过用户界面或 CLI 监控部署状态。
a|* 基于 GitOps 的工作流程
* 跨环境部署更快、更自动、更一致。

|安全管理员
a|* 以代码形式定义和执行安全策略。
* 部署安全工具和配置。
* 审计和管理整个集群的安全状况。
a|* 降低风险，集中合规管理。
* 自动、一致的安全配置。
|===
