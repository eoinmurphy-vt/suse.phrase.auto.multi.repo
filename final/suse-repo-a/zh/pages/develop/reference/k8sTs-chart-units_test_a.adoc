= 图表支持的单位
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE 可观察性

本页包含图表支持的所有单位。对于每个单元，首先会提到该单元的描述，然后是应在 SUSE Observability API 中使用的单元名称。例如，要以 "科学记数法 "显示数值，在公制绑定中使用`sci` 作为单位值。

单位的渲染会根据数值的大小自动调整。例如，如果图表的单位是`bytes` ，该值将被解释为`bytes(IEC)` 。`100` 的值将显示为`100B` ，但`5242880` (`5*1024*1024`) 的值将显示为`5MiB` 。
