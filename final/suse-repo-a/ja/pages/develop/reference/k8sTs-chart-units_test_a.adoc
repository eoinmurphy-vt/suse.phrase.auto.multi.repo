= チャートの対応単位を変更
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

このページには、チャートでサポートされているすべてのユニットが含まれています。すべてのユニットについて、最初にユニットの説明が記載され、その後にSUSE Observability APIで使用するユニット名が続きます。例えば、"Scientific notation"（科学的記数法）で表示される値を得るには、`sci` をメートル法での単位の値として使用する。

単位のレンダリングは、値の大きさに合わせて自動的に調整される。例えば、チャートの単位が`bytes` の場合、値は`bytes(IEC)` と解釈される。`100` の値は`100B` として表示されるが、`5242880` (`5*1024*1024`) の値は`5MiB` として表示される。
