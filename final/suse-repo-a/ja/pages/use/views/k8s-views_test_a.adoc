== 概要

SUSE Observabilityは、Kubernetesとそのコンポーネントに関する深い知識を持っています。クラスタにSUSE Observabilityエージェントをインストールすると、Kubernetesアプリケーションのトポロジを自動的に検出して可視化します。これには、デプロイメント、ポッド、サービス、イングレスなど、アプリケーションを構成するKubernetesリソースが含まれる。また、ノード、ネームスペース、永続ボリュームなど、クラスタを構成するKubernetesインフラのトポロジーを自動的に検出して可視化する。

SUSE Observabilityには、以下のKubernetesネイティブリソース専用の概要とハイライトページがあります：

image::k8s/k8s-menu.png[Kubernetesビュー]

その他のKubernetesリソースはすべて認識され、トポロジービューで可視化される。

== 概要ページ

概要ページは、環境内の特定のKubernetesリソースのハイレベルな概要を提供します。これらの概要では、リソースの健全性、年齢、名前空間、クラスタなど、リソースに関する最も重要な情報が強調表示されます。

image::k8s/k8s-service-overview.png[概要ページ]

概要ページからKubernetesリソースの1つをクリックすると、そのリソースのハイライトページに移動できます。

== ハイライト・ページ

ハイライトページには、特定のKubernetesリソースに関する最も重要な情報の概要が表示されます。ここでは、リソースの健全性、リソース上でアクティブなモニター、リソース上で発生したイベントを確認できます。また、リソースの主要なメトリクスを表示します。

image::k8s/k8s-pod-highlights.png[ハイライト・ページ]
